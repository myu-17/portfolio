宵(照明専門店)

サイトを見る

夜の時間を少し特別なものにする照明ブランド、「宵」のWebサイトを制作しました。
1枚1枚の画像を引き立たせつつ、高級感のあるデザインを目指しました。

制作時間・時期
企画・ワイヤーフレーム / 1週間 デザイン / 2週間 コーディング /1週間 2024年12月に制作。
使用ツール
Illustrator / Photoshop
案件概要
◼︎ハイエンドなデザイン照明を取り扱う、照明専門店。
◼︎コンセプトは「あなたの夜を静かに満たす光」
ターゲット
上質な暮らしを求める、30代後半〜50代後半の男女
クライアントの
課題、要望
①既存のウェブサイトでは、商品やブランドのイメージが十分に伝わっておらず、上質な暮らしを目指すターゲットに響いていない。
②ブランドの強みをアピールできていない。
③照明の光の質感や雰囲気をビジュアルで魅せるための工夫が不足している。
目的
①モダンで洗練された高級感のあるデザインを目指し、ブランドの世界観やイメージを視覚的に伝える。
②ブランドの強みや実際の購入者の声を明確に打ち出し、選ばれる理由をわかりやすく伝える。
③ 照明の光が持つ美しさや空間の雰囲気をビジュアルで丁寧に表現し、商品の魅力を直感的に伝える。
情報設計
ファーストビューでは、余計な要素を極力排除し、照明のもたらす情緒と空間の静けさが際立つように構成。ランプシェードがやわらかく灯る情景にキャッチコピーを重ね、訪問者の印象に残るように設計しました。コンセプトや特長、お客様の声をトップページ内に集約することで、「ただの照明とは違う」という価値を明確に伝え、ブランドへの興味喚起とサイト離脱の防止を狙っています。全体を通して、世界観を壊さずに情報へスムーズに辿れる構成に仕上げました。
デザイン
静けさと光のぬくもりを丁寧に伝えるため、全体を通して時間の移ろいを感じさせる構成に。夕暮れから夜、そして早朝へと変化するグラデーションをページ各所に取り入れることで、光と共に過ごす豊かな時間を想起させる演出としました。また、余白を広めに使い、情報を詰め込みすぎず、洗練された落ち着きのある印象に。暗めの背景は照明の写真を引き立て、滑らかなラインアニメーションを加えることで、光の柔らかさや上質さを視覚的に感じられるよう工夫しています。
