販促バナー(冬季限定チョコレート)

web上のお題を使用した個人制作のバナーです。
冬季限定セールをしている、チョコレートの架空バナーを制作しました。

制作時間・時期
デザイン / 1日 2025年2月に制作。
使用ツール
Illustrator / Photoshop
案件概要
冬季限定の高級一口チョコレートを、特別価格で訴求するバナー広告。
下記リンクのお題を使用させていただきました。
#バナーお題 第1弾 冬限定一口チョコ
ターゲット
20〜40代の働く男性及び女性
クライアントの
課題、要望
◼︎イメージカラー：白、水色
◼︎キャッチコピー：ちょっと一息。贅沢をお裾分け
◼︎割引期間：11月1日〜12月31日まで
◼︎割引価格：10～20%割引
デザイン
高級感と落ち着いた印象を与えるため、オブジェクトや文字の数は最小限に抑え、余白を活かしたレイアウトを意識しました。ごちゃごちゃと情報を詰め込まず、ミニマルな構成にすることで、冬の凛とした空気や静寂感を視覚的に表現しています。 また、視線の流れを意識し、コピー、ビジュアル、セール情報の順に自然に読み進められるよう配置を設計。特に下部のセール情報は帯で明確に切り分けることで、主役のビジュアルを邪魔せず、伝えるべき情報をしっかり届けられるよう工夫しました。
